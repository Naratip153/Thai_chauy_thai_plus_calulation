## 📌 เกี่ยวกับโครงการ

โครงการ **ไทยช่วยไทยพลัส** เป็นโครงการรัฐที่ช่วยอุดหนุนค่าสินค้าอุปโภคบริโภคให้ประชาชน โดยมีเงื่อนไขดังนี้

| รายละเอียด                 | ค่า               |
| -------------------------- | ----------------- |
| รัฐช่วยออก                 | 60% ของราคาสินค้า |
| ประชาชนจ่าย                | 40% ของราคาสินค้า |
| วงเงินสูงสุดต่อวัน (รัฐ)   | ฿200              |
| วงเงินสูงสุดต่อเดือน (รัฐ) | ฿1,000            |
| ระยะเวลาโครงการ            | 4 เดือน           |

---

## ✨ ฟีเจอร์

- **โหมด 1 — รู้ราคาสินค้า**: กรอกราคาสินค้า แล้วระบบจะคำนวณว่ารัฐออกเท่าไหร่ ประชาชนจ่ายเท่าไหร่
- **โหมด 2 — รู้เงินที่มี**: กรอกเงินที่ต้องการจ่าย แล้วระบบจะคำนวณว่าซื้อสินค้าได้รวมกี่บาท
- แสดง **Progress Bar** สัดส่วนรัฐ/ประชาชนแบบ real-time
- แจ้งเตือนเมื่อยอดเกิน **วงเงินสูงสุด ฿200/วัน**
- ปุ่มตัวอย่างราคาสำเร็จรูปสำหรับทดสอบด่วน
- รองรับ **ภาษาไทย** เต็มรูปแบบ (ฟอนต์ Sarabun, รูปแบบตัวเลขไทย)
- **Responsive** — ใช้งานได้บนมือถือและคอมพิวเตอร์

## 🧮 สูตรคำนวณ

### โหมด 1 — รู้ราคาสินค้า

```
รัฐออก   = MIN(ราคาสินค้า × 60%, 200)
ประชาชนจ่าย = ราคาสินค้า − รัฐออก
```

### โหมด 2 — รู้เงินที่ประชาชนมี

```
ถ้าเงินที่มี ≤ 133.33 บาท:
  ซื้อได้รวม = เงินที่มี ÷ 40%
  รัฐออก    = ซื้อได้รวม × 60%

ถ้าเงินที่มี > 133.33 บาท (รัฐออกเต็มวงเงิน):
  รัฐออก    = 200
  ซื้อได้รวม = เงินที่มี + 200
```
